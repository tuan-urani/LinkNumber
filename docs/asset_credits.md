# Asset Credits

Use this document to track all third-party assets used in the project.
Do not add external assets into the app unless the license is clear and compatible.

## Accepted Licenses

- `CC0`
- `MIT`
- Commercial license with valid purchase/subscription

## Third-Party Asset Register

| Asset Path | Source URL | Author/Publisher | License | Download Date | Proof (invoice/screenshot/link) | Attribution Required |
| --- | --- | --- | --- | --- | --- | --- |
| `assets/game/balls/<file_name>` |  |  |  |  |  | Yes/No |
| `assets/game/effects/explosion/<file_name>` |  |  |  |  |  | Yes/No |

## Notes

- If attribution is required, add it to app credits or README before release.
- Keep original downloaded package/license file in a safe internal location.
- Prefer `snake_case` filenames that match Figma/export naming.
