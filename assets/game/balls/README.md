# Balls Assets

Place third-party ball assets in this folder.

## Naming

- Use `snake_case`.
- Keep names meaningful and stable, for example:
  - `ball_red.png`
  - `ball_blue_outline.webp`
  - `ball_gloss_highlight.png`

## Before Adding

- Confirm license compatibility.
- Add source and license details in `docs/asset_credits.md`.
