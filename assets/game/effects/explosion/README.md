# Explosion Effects Assets

Place third-party explosion/effect assets in this folder.

## Naming

- Use `snake_case`.
- Suggested naming patterns:
  - `explosion_burst_01.png`
  - `explosion_smoke_01.png`
  - `explosion_sheet_01.png`

## Before Adding

- Confirm license compatibility.
- Add source and license details in `docs/asset_credits.md`.
